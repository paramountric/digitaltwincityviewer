# Speckle server

The Speckle server is available at http://localhost:8080.

The Speckle server is used to store and manage the data for the DTCV Platform.

## GraphQL API

The Speckle GraphQL API is available at http://localhost:8080/graphql.

The Speckle GraphQL API is used to query and mutate the data for the DTCV Platform.
